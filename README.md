# TFG - WarehouseDT

Diseño y Desarrollo de un Gemelo Digital para la Gestión de Almacenes de PYMEs. Junio 2026

## Funcionalidades

* Gestión de almacenes, productos y stock.
* Gestión de usuarios y roles.
* Asignación inteligente de tareas a operarios.
* Gemelo digital 3D en tiempo real.
* Registro de movimientos mediante NFC y códigos de barras.
* Aplicación móvil para operarios.
* Sincronización en tiempo real mediante WebSockets.
* Verificación de cuentas y recuperación de contraseña por correo electrónico.

## Tecnologías

* **Backend:** FastAPI, SQLAlchemy, PostgreSQL
* **Frontend:** React, Vite
* **Móvil:** React Native, Expo
* **Gemelo digital:** Unity WebGL
* **Despliegue:** Railway
* **Correo electrónico:** Resend
